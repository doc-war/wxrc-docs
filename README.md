# 文档说明


WXRC系统将微信机器人能力和业务系统做物理隔离，类似支付通道定位，连接业务系统（客户端）和微信服务端（服务端），主要提供如下功能：

请求前置：
* 对请求来源（chId、botcode）和签名（sign）进行合法性判断
* 对来源报文统一日志记录
* 对来源报文进行解析路由

业务处理：
* 对客户端业务请求进行业务处理和响应，向微信服务端下发机器人客户端消息
* 对微信消息回调进行解析和业务处理，并将有效消息转发回前台业务系统
